      - Stores client's publicKey in the peer store via store.upsert()
      - Returns PairResponse: { serverPublicKey, relayUrl }
    - InviteStore is a simple Map<string, InviteEntry> (token -> { expiresAt, serverPublicKey, used })

    - Both routers registered in createApp() with shared InviteStore instance

    - 4+ tests for pair route (happy path, invalid token, expired token, bad key format)
